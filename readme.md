# Web Debugger
An app for viewing the RAW HTML code of a web page on iOS.

![](Resources/previrew.gif)

Features:
- basing browser with navigation
- HTML parser tool
- activity logging
- cookies preview
- searching in HTML noodes and cookies
- dark theme support
